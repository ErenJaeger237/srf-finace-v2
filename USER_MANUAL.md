# SRFM User Manual - Street Ride Finance Manager

Welcome to the SRFM platform. This tool ensures complete transparency and financial integrity for the Street Ride crew.

## 1. Accessing the System
- **Login:** Enter your organizational email and private Access Key.
- **Onboarding:** Upon your first login, a guided carousel will introduce you to our cell-based budget structure and reporting features.

## 2. Dashboard Interface
The Main Dashboard (Bento Grid) provides an instant snapshot of your unit's health:
- **Available Balance:** Real-time visibility into organization liquidity.
- **Cell Budgets:** Interactive cards allowing you to monitor spending per unit (Logistics, Marketing, Security).
- **Audit Trail:** An immutable log of all financial movements for complete accountability.

## 3. Financial Management
Authorized Finance Managers can perform the following via the **+ New Entry** action menu:
- **Record Contribution:** Log member payments toward organization goals.
- **Record Expense:** Deduct spending from specific cell budgets.
- **Record Income:** Log revenue generated by specific cells (e.g., event earnings).
- **Create New Cell:** Establish new operational budgets as the organization scales.

## 4. Reports & Transparency
Financial integrity is a core pillar. 
- You can generate a comprehensive **Financial Report** at any time.
- Navigate to the **Audit Trail** screen and tap the **Export (Download)** icon.
- This creates an Excel file containing three distinct sheets: **Contributions**, **Expenses**, and **Budget Summary**.
