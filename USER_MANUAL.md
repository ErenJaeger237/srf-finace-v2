# SRFM User Manual

## Login
- Use your organization email and the provided Access Key.
- First-time users will see the onboarding carousel explaining the platform.

## Dashboard (Finance Hub)
- **Available Balance:** Real-time view of your current liquid assets.
- **Bento Grid:**
    - **Income/Expenses:** Quick overview of totals.
    - **Cell Budgets:** Horizontal scroll list to monitor individual unit spending.
    - **Recent Transactions:** Latest activities.

## Features
- **Members List:** View contribution status of all 14 members.
- **Budgets:** Track budget utilization by cell.
- **Audit Trail:** Review historical actions.
- **Reporting:** Tap the download icon in the Audit Trail to generate and share a full financial report (Excel).

## Actions
- Use the central **+ New Entry** button to:
    - Record a contribution.
    - Log a cell expense.
    - Record cell-generated income.
    - Create new operational cells.
