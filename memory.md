# Project Memory: Street Ride Finance Manager (SRFM)

## ✅ COMPLETED (Frontend - Flutter)
- **Project Initialization:**
    - Workspace cleaned of old Vue.js files.
    - Initialized Flutter project targeting Android 6+ and iOS 17+.
- **Theme & Identity:**
    - Font: **Plus Jakarta Sans** (Headlines) & **Inter** (Body).
    - Palette: Charcoal background (`0xFF0D0D0D`), Street Orange (`0xFFFF4500`), Success Teal, and Error Red.
- **Glassmorphism Engine:**
    - Created `GlassCard` component using `BackdropFilter` (Blur) and `LinearGradient` (Fallback).

- **Screens & UI Implementation:**
    - **Onboarding:** Skateboard-themed 3-step flow. Persistent storage via `shared_preferences`.

    - **Main Dashboard:** High-fidelity **Bento Grid** layout.
    - **Members:** 14-member list with status tracking (Paid/Pending).
    - **Budgets:** Detailed cell breakdown with progress indicators.
    - **Audit Trail:** Transaction log with category filtering.
- **Role-Based Access (RBAC):**
    - **Admin:** Views full group charts (Pie/Bar) and CRUD actions.
    - **Member:** Views personalized "My Contribution" card and view-only data.
- **Data Visualization:**
    - Integrated `fl_chart`.
    - **Expense Distribution:** Pie chart by cell.
    - **Group Contributions:** Bar chart for all 14 members.
    - **Income Comparison:** Bar chart comparing revenue generated by cells.
- **Export Engine:**
    - `ExportService` implemented using `excel` and `share_plus`.
    - Generates multi-sheet report (Contributions, Expenses, Budget Summary).
- **Forms:**
    - Glassmorphic bottom-sheets for **Add Contribution**, **Record Expense**, **Record Cell Income**, and **Create New Cell**.

## ✅ COMPLETED (Backend - Node.js)
- **Environment & Runtime:**
    - Initialized Node.js (v20) with TypeScript.
    - Configured as **ECMAScript Module (ESM)** for modern compatibility.
    - Used **tsx** for reliable development execution.
- **Database Architecture:**
    - **Prisma 6.2.1.**
    - **Schema:** Multi-tenant design with `organizationId` on all tables (Organization, User, Cell, Contribution, Expense, AuditLog).
- **Security & Middleware:**
    - **JWT Authentication:** Implemented `authenticate` middleware with secure `JWT_SECRET`.
    - **RBAC:** Implemented `isAdmin` middleware to restrict write operations to the Finance Manager.
    - **bcrypt:** Password hashing for user security.
- **API Endpoints Implemented:**
    - `POST /api/auth/login`: Credential verification and JWT issuance.
    - `GET /api/members`: Multi-tenant listing of organization members.
    - `GET/POST /api/cells`: Cell management with budget and income tracking.
    - `GET/POST /api/expenses`: Transactional spending records with automated audit trail.
    - `GET/POST /api/contributions`: tracking member payments.
    - `GET /api/audit`: Chronological action log retrieval.
- **Database Seeding:**
    - Idempotent `prisma/seed.ts` using `upsert`.
    - Auto-generates organization "Street Ride".
    - Initializes sample Cells: Logistics, Marketing, Security.

## 🧱 Data Model Details
| Table | Description |
|---|---|
| **Organization** | Primary tenant. |
| **User** | Roles: `ADMIN`, `MEMBER`. Multi-tenant. |
| **Cell** | Tracks `budget`, `spent`, and `income` per unit. |
| **Contribution** | Member-specific payments. |
| **Expense** | Cell-specific spending. |
| **AuditLog** | Immutable log of all sensitive actions. |
